Referencing Elements ../ There might be circumstances when elements have common attributes with no differences. This can be distress when updating all these attributes for every element. And this is where referencing can be applied.

First, pick a reference element where the attributes can be referenced from. Then, add an id attribute to it (if there is none). It can be given any id, so long as it is unique from the rest of the page.

<p id="text" stye="color: red">Hello world</p>

Then, for every items that references to it, just set their ref-id attribute to the id. The snippet below demonstrates that simply.

{{showPreviewCode('referencing-1')}}


The two elements then copies all attributes (except the id) from the reference element...

From <p ref-id="text">Lorem Ipsum</p>,
to <p style="color: red">Lorem Ipsum</p>

Only Specific Attributes
To specify which attributes must only be included, use the ref-only attribute. There, list down the to-copy attributes, separated by comma ,.
