Snippets ../ Take a look at this code:

<snippet src="other.html"></snippet>

It simply imports other.html right there, directly. No iframes. No embeds. Simply, Snippets.

And in case only specific elements are needed, there are two ways to do so.

1. With an id
Specify a desired element by setting the snippet's snip-id attribute to the id.

<snippet src="other.html" snip-id="mytext"></snippet>

The snippet above shall only display the element in other.html that has an id mytext.

2. With a CSS Selector
Also, CSS Selectors (just like the ones used in querySelector) can be used for picking specific elements. Just set the selector attribute to the selector.

The example below shall display all span elements from other.html

<snippet src="other.html" selector="span"></snippet>

Snippet Variables
Snippets has support for variables as well.

These variables are enclosed by percent symbols: %variableName%, which is placed anywhere inside the target file (at the example above, other.html).

These variables can then be substituted with specific values by adding the value-variableName attribute to the snippet element, with its value as the variable's value.

For example, with an attribute value-text="Hello World", all %text% in the target file are substituted to Hello World.

<snippet src="other.html" selector="span" value-text="Hello World"></snippet>
<span style="color: blue">%text%</span>
Hello World
