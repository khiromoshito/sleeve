Page Routing ../ Page Routing is used for navigating to a different page without reloading the current tab.

To enable, add the page-routing attribute to the anchor element.

<a href="somepage.html" page-routing>Go there</a>

When the link is clicked, Sleeve then takes the target somepage.html, updates the location, and displays the page in the same display without reloading.
