Iterating Elements ../ One of Sleeve's power features is scalability. And following upon this are Iterating elements. These are simply elements that can clone itself corresponding to an index or a list.

To make an element iterating, just add the iterating attribute.

Iterating Lists
Using i-count and %i% is best for explicit cloning which does not require reference. But in the case of lists (aside from the given above), there is a shorthand for handling iterations.

The i-items attribute can be used. Its value follows the following format:

item : list

The item corresponds to the variable name for every iterations (items on the list), and the list represents the array variable itself. item can then be used with a variable placeholder (surrounded with %): %item%.
