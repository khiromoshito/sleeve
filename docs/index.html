Introduction ./ Sleeve is a front-end web framework for building highly flexible webpages. It acts like an extension to HTML, with <semantic elements> and special attributes for crafting dynamic user interface.

This is a newly-developed framework. Its first commit was on February 15, 2020, and until now, this project is still under progressive development. Sleeve is open source, and you can contribute to the codebase as well. The full project repository (including this documentation) can be found here.

Sleeve has been developed to help overcome difficulties in developing reactive, responsive web pages. Sleeve is composed of two components that build up the whole framework:

CORE
For the DOM manipulations, state management and routing
Sleeve-Suit (User Interface)
For widgets, semantics and responsive layout


The core component provides features for structuring the page, while Suit manages the design. Both execute for a dynamic web application.

Sleeve also has a powerful DOM manipulation, and most of it does not require writing JavaScript. Sleeve core makes use of the HTML element itself to host DOM manipulations. Just adding a simple attribute or two, and the element becomes alive.

Sleeve's approach aids web development to be faster, more efficient, flexible and scalable. Powerful as it can be, Sleeve supports JavaScript-ing directly in HTML elements (core) and even in CSS (Sleeve-Suit). And there's more of the features to know of. Sleeve is simply made to become productive.
